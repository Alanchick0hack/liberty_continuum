# DEFINITION — Non-Capture

## Definition
Non-capture is the property that a system cannot be quietly seized by concentrated power, whether through money, political access, data control, model control, or infrastructure monopolies.

## Required properties
- Transparent governance and procurement
- Audit independence
- Interoperability and replaceability
- Anti-monopoly protections for critical rails
- Public visibility into incentives and conflicts

## Not this
Non-capture is not:
- One vendor
- One model
- One gatekeeper
- One unchallengeable authority
