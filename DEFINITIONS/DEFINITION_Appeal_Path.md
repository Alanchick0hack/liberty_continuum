# DEFINITION — Appeal Path

## Definition
An appeal path is a guaranteed process for a person to challenge an impactful decision, correct inputs, contest rule application, and obtain review within defined time bounds.

## Required properties
- Clear entry point for appeal
- Time bounds for response
- Ability to contest inputs
- Ability to contest rule application
- Human review lane for unresolved disputes
- Written outcome with reason reference

## Not this
An appeal path is not:
- A dead-end support form
- A discretionary “maybe” review
- A process that requires wealth to survive
