# PROTOCOL — Appeals

## Purpose
Guarantee meaningful recourse without requiring wealth, influence, or insider access.

## Required lanes
- Automated correction lane for input errors
- Review lane for rule disputes
- Human review lane for unresolved cases

## Required guarantees
- Clear entry point
- Clear time bounds
- Written reason reference for outcomes
- Ability to submit counter-evidence
- Ability to escalate to human review

## Prohibited patterns
- Discretion-only review
- Non-response default denial
- Appeals that require payment to function

## Public principle
If it impacts your life, you must be able to see it.
