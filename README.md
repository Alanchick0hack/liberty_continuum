## 📂 Included Documents and Definitions

The release contains important documents:

- **VISION.md** explains the purpose and goals.  
- **SCOPE.md** defines what liberty_continuum covers.  
- **CANONICAL_INDEX.md** and **INDEX.md** help you find key topics.  

You will also find clear definitions for terms like:

- **Auditability:** Being able to track and review decisions.  
- **Appeal Path:** A process to challenge decisions.  
- **Rights Layer:** The set of core rights protected.  
- **Minimal Coercion:** Limiting forced actions or controls.  
- **Provenance:** Tracing the origin of data or actions.  
- **Governance as Infrastructure:** Treating governance systems like basic public services.  

These help you understand how the software supports transparent and fair management of AI and civil rights.
